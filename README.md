# 🎬 Netflix Data Analysis

## Project Overview
This project analyzes Netflix Movies and TV Shows data using Python, Pandas, SQL, SQLite, and Data Visualization.

## Tools Used
- Python
- Pandas
- NumPy
- SQLite
- Matplotlib
- Jupyter Notebook

## Dataset
Netflix Movies and TV Shows Dataset

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Movies vs TV Shows Analysis
- Content Growth Analysis
- Genre Analysis
- Country Analysis
- SQL Analysis

## Key Insights
- Movies dominate Netflix's content library.
- Netflix experienced rapid growth between 2016 and 2020.
- TV-MA is the most common rating.
- The United States contributes the largest amount of content.
- International Movies and Dramas are among the most popular genres.
